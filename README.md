# WebApp

## Firebase-Konfiguration
Aus Sicherheitsdaten sind keine echten Firebase-Zugangsdaten im Repo enthalten. Sollte ich welche übersehen haben, bitte ich damit verantwortungsbewusst umzugehen.

Ein Link zu einer LiveDemo: https://scotty2404.github.io/PetCheck/login 

Um das Projekt lokal auszuführen:
1. Erstelle PetCheck\src\environtment.ts nach dem schema der environment.example.ts und trage eigene Firebase-Konfiguration ein
2. Erstelle PetCheckBackend\serviceAccountKey.json mit eigenen service account Daten.
3. führe das frontend aus mit ng serve oder erstelle einen build
4. Starte den Backend server mit node server.js um push notifications zu erhalten
